# Memory Match Game

A fun memory matching game built with HTML, CSS, and JavaScript. Match pairs of cards to progress through levels!

## Features

- 6 different card types (colors by default, or custom photos)
- Progressive levels: Start with 6 cards, each level adds 2 more cards
- Randomized card layouts for unique playthroughs
- Full-screen UI with no scrollbars
- Camera integration to capture custom photos for cards
- Compatible with GitHub Pages deployment

## How to Play

1. On the title screen, optionally click on the photo cards to take pictures using your camera.
2. Click "Start Game" to begin.
3. Click cards to flip them and find matching pairs.
4. Match all pairs to advance to the next level.
5. Each level increases the number of cards by 2.

## Deployment

This game is designed to be deployed on GitHub Pages:

1. Push this code to a GitHub repository.
2. Go to the repository settings.
3. Under "Pages", select the main branch as the source.
4. The game will be available at `https://yourusername.github.io/repositoryname/`.

## Browser Requirements

- Modern browser with camera support for photo feature
- HTTPS required for camera access (automatically provided by GitHub Pages)
